# Else-If Statement
Else-if statement is just a simple way to add a condition within your else statement. Let's say we make a program where we display what vowels a user enters

```c++
	char vowel;

	cout << "Enter a vowel: ";
	cin >> vowel;

	if(vowel == 'a') {
		cout << "You entered letter a";
	} else if(vowel == 'e') {
		cout << "You entered letter e";
	} else if(vowel == 'i') {
		cout << "You entered letter i";
	} else if(vowel == 'o') {
		cout << "You entered letter o";
	} else if(vowel == 'u') {
		cout << "You entered letter u";
	} else {
		cout << "You entered a consonant";
	}

	cout << endl;
```
